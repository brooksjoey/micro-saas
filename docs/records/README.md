# Implementation Records

This directory contains implementation records for all completed tasks. Each record documents:
- What was built
- Files modified
- Technical implementation details
- ADR alignment
- Tests added
- Operational considerations

## Naming Convention

```
{YYYY-MM-DD}-{short-key}.md
```

Example: `2025-01-15-auth-billing-migration.md`
